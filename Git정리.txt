committed - 데이터가 로컬 데이터베이스에 안전하게 저장된 상태
Modified - 수정한 파일을 로컬 데이터베이스에 커밋하지 않음 상태
staged - 현재 수정한 파일을 곧 커밋할 것이라고 표시한 상태

Git flow 
git stash
- 현재 작업 상태를 백업: git stash save
- 새로운 작업 진행 및 완료
- 이전 작업 상태로 되돌림: git stash pop
